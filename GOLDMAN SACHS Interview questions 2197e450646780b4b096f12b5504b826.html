    Given an array of integers arr[] representing non-negative integers, arrange them so that after concatenating all of them in order, it results in the largest possible number. Since the result may be very large, return it as a string.

INTUITION : Understanding of lexicographical sort is required because ultimately we want that every character should be sorted

lexo sort : for “10” and “1” it will compare 10+1 and 1+10 which is bigger and return the bigger one hence it always give a output such that if first char is same it will compare the second char, third char so on.

// User function template for C++
class Solution {
public:
string findLargest(vector<int> &arr) {
// code here
vector<string> arr1;


for(int ele: arr) {
    arr1.push_back(to_string(ele));
}
// custom comparator to sort the string lexicographically 
    sort(arr1.begin(),arr1.end(),[](const string &x,const string &y){
     return (x+y)>(y+x);
    });

    if(arr1[0]=="0")return "0";


    string res="";
    for(auto i : arr1)
    {
        res+=i;
    }

    return res;
}
};
