Initializing a Spring Boot Application

Spring Boot is a popular framework for building Java-based applications. It simplifies the setup and development process by providing default configurations and reducing boilerplate code.

Prerequisites

Before you start, ensure you have the following installed:

  • Java Development Kit (JDK): Spring Boot requires Java. Make sure you have JDK 8 or later installed.
  • Build Tool: You can use either Maven or Gradle as your build tool.
  • Integrated Development Environment (IDE): While not mandatory, using an IDE like IntelliJ IDEA, Eclipse, or Spring Tool Suite (STS) can enhance your development experience.

Setting Up a Spring Boot Project

Using Spring Initializr

Spring Initializr is a convenient web-based tool to generate a Spring Boot project structure with the dependencies you need.

  1. Visit Spring Initializr: Go to start.spring.io.
  1. Project Metadata:
    • Project: Choose either Maven or Gradle as your build tool.
    • Language: Select Java.
    • Spring Boot: Choose the latest stable version.
  1. Project Metadata:
    • Group: Enter a group ID (e.g., com.Ke.aerospace.aircrafts.hue) This hierarchical structure helps in organizing projects under specific product lines and modalities..
    • Artifact: Enter an artifact ID (e.g., order-management-service)This artifact ID represents the specific project "Preferences" within the Hue product line..
    • Name: Provide a name for your application.
    • Description: Optionally, add a description.
    • Package Name: Enter the base package name.
    • Packaging: Choose between JAR (default) or WAR.
    • Java: Select the Java version you are using.
  1. Dependencies:
    • Click on "Add Dependencies" and select the necessary dependencies for your project. Common starters include:
      • Spring Web: For building web applications.
      • Spring Data JPA: For database interactions using JPA.
      • Spring Security: For securing your application.
      • H2 Database: For an in-memory database useful for testing.
  1. Generate the Project:
    • Click on "Generate" to download the project as a ZIP file.
    • Extract the ZIP file and import the project into your preferred IDE.

  • Eclipse:
    1. Open Eclipse and select "File" > "New" > "Other."

    1. Open STS and select "File" > "New" > "Spring Starter Project."
    1. Follow the prompts to configure your project.

Project Structure

Once your project is set up, you'll see a standard directory structure:

  • src/main/java: Contains your Java source files.
  • src/main/resources: Contains configuration files like application.properties.
  • src/test: Contains test cases.
  • pom.xml or build.gradle: Configuration file for Maven or Gradle, respectively.

Running the Application

To run your Spring Boot application, you can use the following methods:

  • Using Maven: Navigate to the project directory in your terminal and run:
    ./mvnw spring-boot:run
  • Using Gradle: Navigate to the project directory in your terminal and run:
    ./gradlew bootRun
  • Using IDE: Right-click on the main application class (annotated with @SpringBootApplication) and select "Run."
